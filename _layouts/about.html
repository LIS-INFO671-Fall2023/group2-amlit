Title

A free text name given to the resource. If the resource lacks a title, use a summarized description of the item.

Creator

Creator of the resource. If the creator is not clear, list the agency that funded or housed the original resource.

Date that the resource was created. Use ISO 8601 format. If the date created is not available, use the copyright date or date the original resource was added to the holding institution.

Description

A free text description of the resource. Description should provide more detail than given in the resource title. Avoid descriptions that give “interpretation” of the resource. State the facts.

Subjects

Subject headings or keywords that describe the resource. These should reflect key aspects and persons involved or mentioned in the resource. Use Library of Congress Subject Headings.
